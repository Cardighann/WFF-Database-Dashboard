# WFF-Database-Dashboard
Webster Frozen Foods had an outdated system for storing information about their orders, products, and prices. I designed a full relational database to store all of the required information, and supplied an interactive dashboard to monitor and alter the information in the database.

## VIEW THE DASHBOARD
To view the finished product, please see the folder "Dash-Images". In this folder you will find screenshots of all relevant sections of the dashboard. I have chosen this method due to lack of sharing abilites of Oracle Apex. 

## VIEW THE SQL
To see my SQL code, please see "DataWarehouse.sql", "WFF_DATA.sql", "WFF_Script" for all of the code used to create this relational database.

I made an ERD and a Relational Diagram for this project, images of both of these are within the "WFF_DIAGRAMS" folder.
